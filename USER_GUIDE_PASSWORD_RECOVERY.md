# How to Recover Your Password - User Guide

## Quick Start (3 Easy Steps)

### Step 1: Enter Your Email
1. Click **"Forgot password?"** on the login page
2. Enter the email address associated with your account
3. Click **"Send Verification Code"**

### Step 2: Enter the Code
1. Check your email for a 6-digit code
2. Enter it on the same page
3. The form auto-submits when you've entered all 6 digits
4. You'll see a checkmark when successful

### Step 3: Set a New Password
1. You'll be redirected to the password reset page
2. Enter your new password (at least 8 characters)
3. Confirm your new password
4. Click **"Reset Password"**
5. You're done! Log in with your new password

**Total Time: ~45 seconds**

## Common Questions & Answers

### Q: I don't see the verification code email
**A:** Check these places:
1. **Inbox** - Look for email from "noreply@supabase.io"
2. **Spam/Junk folder** - Check spam filters
3. **Promotional tab** - On Gmail, check this tab
4. **Wait a moment** - Email can take 30-60 seconds to arrive

**If still not received:**
- Click **"Resend in 60s"** to request a new code
- Check your email address is correct
- Try a different browser or device
- Contact support if persistent issues

### Q: The code I entered is wrong
**A:** This can happen for a few reasons:
1. **Typo** - Double-check each digit carefully
2. **Expired** - Code only works for 10 minutes, get a new one
3. **Already used** - Each code works only once
4. **Wrong code** - Make sure you copied correctly

**Solution:** Click **"Resend in 60s"** to get a fresh code

### Q: I've tried 5 times and it keeps saying invalid
**A:** You've hit the security limit. This is to protect your account:
1. **Wait** - Come back in a few minutes
2. **Request new code** - Click "Resend" when timer expires
3. **Check email** - Verify you're copying the code correctly
4. **Clear browser cache** - Sometimes helps with auto-fill issues

### Q: Can I change my email during recovery?
**A:** Yes! On the OTP entry screen:
1. Click **"Change Email"** button
2. You'll return to the email entry screen
3. Enter a different email address
4. Request a new code

### Q: The page doesn't work on my phone
**A:** Try these steps:
1. **Refresh page** - Pull down to refresh
2. **Clear cache** - In browser settings
3. **Use different browser** - Try Chrome, Safari, or Firefox
4. **Check internet** - Make sure you have good connection
5. **Disable extensions** - Ad blockers can interfere

### Q: What if I didn't request this code?
**A:** Someone may have tried to access your account:
1. **Don't share the code** - Never give it to anyone
2. **Ignore it** - Code expires in 10 minutes anyway
3. **Change password** - Use our recovery process to set a new one
4. **Enable 2FA** - Consider enabling two-factor authentication

### Q: My new password isn't saving
**A:** Check these things:
1. **Password too short** - Must be at least 8 characters
2. **Passwords don't match** - Confirm passwords must be identical
3. **Special characters** - Avoid very unusual characters
4. **Clear cache** - Try again after clearing browser data

### Q: I'm locked out completely
**A:** Don't worry, you can still recover:
1. **Try this process** - Use password recovery from login page
2. **Verify your email** - Make sure email is correct
3. **Wait 30 minutes** - Security lockouts are temporary
4. **Contact support** - If persistent, reach out to our support team

## Security & Privacy

### We Take Security Seriously

**What we do to protect you:**
✓ Verification codes are encrypted
✓ Codes expire after 10 minutes
✓ Each code works only once
✓ We limit attempts to prevent brute force
✓ Your password is never shared
✓ Connection is always encrypted (HTTPS)

**What we don't do:**
✗ Store your password as plain text
✗ Share your email with third parties
✗ Send unsolicited emails
✗ Access your account after reset
✗ Store your verification codes

### Tips for Strong Passwords

A good password:
- ✓ At least 8 characters (12+ is better)
- ✓ Mix of uppercase and lowercase
- ✓ Numbers and symbols (!@#$%^&*)
- ✓ Not your username or email
- ✓ Not a dictionary word
- ✓ Unique to this account

**Example of strong password:**
```
Mz#Butterfly7!Quest2024
```

**Example of weak password:**
```
123456  (too simple)
password  (too common)
Lifebook2024  (predictable)
```

## Troubleshooting

### Issue: "Failed to send OTP. Please try again."

**Possible causes:**
1. Email address doesn't exist in our system
2. Server connection problem
3. Email service temporarily unavailable

**Solutions:**
1. Verify email is correct
2. Check your internet connection
3. Wait a minute and try again
4. Try a different email if registered with multiple

---

### Issue: "Invalid or expired OTP"

**Possible causes:**
1. Code already used once
2. Code expired (10-minute limit)
3. Typo in code entry
4. Wrong code entered

**Solutions:**
1. Click "Resend" to get a new code
2. Copy code directly from email (avoid typos)
3. Don't wait too long - code expires quickly
4. Request new code if needed

---

### Issue: "Too many failed attempts"

**Possible causes:**
1. Multiple wrong codes entered
2. Security protection activated
3. Account under verification hold

**Solutions:**
1. Wait 5-10 minutes
2. Click "Resend" to get a fresh code
3. Try again with the new code
4. Contact support if continues

---

### Issue: Browser says "This page can't be reached"

**Possible causes:**
1. No internet connection
2. Browser cache corruption
3. JavaScript disabled
4. VPN/proxy issues

**Solutions:**
1. Check internet connection
2. Refresh the page (Ctrl+R or Cmd+R)
3. Clear browser cache (Settings → Clear browsing data)
4. Disable VPN or proxy temporarily
5. Try different browser
6. Disable extensions
