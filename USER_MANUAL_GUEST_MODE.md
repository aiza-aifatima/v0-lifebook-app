# Lifebook User Manual - Guest Mode Edition

Welcome to Lifebook! This guide will help you get started and make the most of your personal development journey.

## Getting Started

### Your First Visit

1. **Open the App**
   - Visit the Lifebook app (local or deployed URL)
   - You'll see a beautiful welcome screen

2. **Enter Your Name**
   - Type your name in the "Your Name" field
   - Must be 2-50 characters
   - Click "Start Your Adventure"

3. **You're In!**
   - You'll be taken to the Dashboard
   - Your session is automatically saved
   - You're ready to start your journey

### Returning to the App

- Simply visit the app again
- Your guest session automatically loads
- All your data is right where you left it
- No login needed!

## Data Management 💾

### Backing Up Your Data

**Why Backup?**
- Protect against accidental data loss
- Switch devices seamlessly
- Keep multiple versions
- Peace of mind

**How to Backup:**
1. Go to Profile
2. Scroll to "Data Management"
3. Click "Export Data" button
4. JSON file downloads automatically
5. Save it somewhere safe!

**File Details:**
- Named: `lifebook-backup-YYYY-MM-DD.json`
- Contains: All your data
- Format: Plain text JSON
- Size: Usually 100-500 KB

### Restoring Data

**When to Restore:**
- You cleared browser data by mistake
- You're on a new device
- You want to recover old data
- You're switching browsers

**How to Restore:**
1. Go to Profile
2. Scroll to "Data Management"
3. Click "Import Data" button
4. Select your backup file
5. Confirm the action
6. App reloads with your data

**Important:** This replaces current data!

### Switching Devices

**From Device A to Device B:**
1. On Device A:
   - Go to Profile
   - Export your data
   - Save file (email, cloud, USB)

2. On Device B:
   - Open Lifebook
   - Create guest session
   - Go to Profile
   - Import your backup file
   - All your data appears!

### Clearing Everything

**To Start Fresh:**
1. Go to Profile
2. Scroll to "Danger Zone"
3. Click "Clear All Data"
4. Confirm you're sure (important!)
5. All data permanently deleted
6. Restart with new session

**⚠️ Warning:** This cannot be undone! Backup first if unsure.

## Troubleshooting 🔧

### "My Data Disappeared!"

**What to do:**
1. Check if it was cleared accidentally
2. Restore from backup if available
3. Check browser storage settings
4. Clear cache and try again

**Prevention:**
- Backup regularly
- Don't clear browser data
- Keep backup files safe

### "Can't Export Data"

**Try this:**
1. Check file size (should be small)
2. Ensure JavaScript is enabled
3. Try different browser
4. Check browser permissions

### "Import File Won't Load"

**Check:**
1. File is valid JSON (not corrupted)
2. File size is reasonable (<5MB)
3. Browser allows file upload
4. Try exporting a test file first

### "Data Not Saving"

**Check:**
1. Browser allows localStorage
2. You have storage space
3. JavaScript is enabled
4. No browser extensions blocking

### "Lost Session"

**Reasons:**
- Browser cleared data
- Incognito mode (clears on close)
- Storage full
- Browser issue

**Recovery:**
- Restore from backup
- Check browser storage
